## A High‑Performance Rule‑Based Multi‑Agent Framework for Spatio‑Temporal Vector Dynamics: Application to *Anopheles stephensi* in Somali Woreda, Ethiopia


### 1. Abstract

**Background**: Vector‑borne diseases (VBDs) remain a major public health challenge in Africa, 
exacerbated by rapid urbanisation and the spread of invasive vectors such as *Anopheles stephensi*. 
Data scarcity in resource‑limited settings hinders traditional modelling approaches.
**Methods**: We present a novel, high‑performance rule‑based Multi‑Agent Simulation (MAS) framework 
called MonadVSIM(Monad Vector Simulator) that integrates heterogeneous environmental layers (temperature, 
precipitation, building density, population density) with a temperature‑dependent Metzler matrix life‑cycle 
model for vectors. The architecture uses a concurrent spatial registry with StampedLock optimisation and an 
object‑pooling lifecycle manager to simulate large agent populations on standard hardware. We validate the 
framework using a 3 months (60 Days) simulation covering the entire Somali woreda, and utilize the Boyce
index the assess the accurary of the data generated by the simulator.

**Keywords**: multi‑agent systems, vector‑borne disease, *Anopheles stephensi*, Metzler matrix, 
high‑performance computing, spatial ecology.
